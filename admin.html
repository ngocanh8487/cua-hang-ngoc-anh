📖 Cách dùng (3 bước, không cần biết code):
  1. Sửa thông tin bên dưới: đổi giá, đổi tên, thêm hoặc xóa sản phẩm...
  2. Bấm nút xanh "⬇ Tải xuống file dữ liệu mới" ở cuối trang → máy tải về file tên products.js.
  3. Chép file vừa tải đè vào file cũ trong thư mục website-ban-hang → js (chọn Replace/Thay thế) → mở website bấm Ctrl+F5 là thấy thay đổi.

💡 Thay mã QR: chép ảnh QR mới đè lên qr-vn.png (Việt) hoặc qr-lao.png (Lào). Thay ảnh sản phẩm: chép ảnh vào thư mục images rồi gõ tên file vào ô "Ảnh" của sản phẩm (ví dụ: images/gao.jpg).
