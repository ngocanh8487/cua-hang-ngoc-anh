📖 Cách dùng (không cần biết code):
  1. Sửa thông tin bên dưới: đổi giá, thêm/xóa sản phẩm, thêm/xóa danh mục...
  2. Bấm nút xanh "⬇ Tải xuống file dữ liệu mới" ở cuối trang → máy tải về file tên products.js.
  3. Vào trang github.com/ngocanh8487/cua-hang-ngoc-anh (đăng nhập nếu được hỏi) → bấm Add file → Upload files → kéo file products.js vừa tải vào → bấm nút xanh Commit changes.
  4. Chờ 1–2 phút → mở website bấm Ctrl+F5 là thấy thay đổi.

💡 Thay mã QR hoặc thêm ảnh sản phẩm: cũng vào Add file → Upload files trên GitHub, tải ảnh lên (QR Việt đặt tên qr-vn.png, QR Lào qr-lao.png, ảnh sản phẩm đặt tên không dấu ví dụ gao.jpg rồi gõ tên đó vào ô "Ảnh" của sản phẩm bên dưới).
